Read a _UNICODE_STRING at us_vaddr and return the raw byte buffer.

Layout (x64): +0x00 Length u16 byte count of string data +0x02 MaximumLength u16 +0x04 _padding u32 +0x08 Buffer u64 pointer to UTF-16LE data
