Module symlinks 

Source
Expand description

Windows kernel symbolic link object scanner.

Enumerates symbolic link objects in the Windows Object Manager namespace by walking from ObpRootDirectoryObject and filtering for objects whose type name is “SymbolicLink”. Symlinks map DOS device names (\DosDevices\C:) to NT device paths (\Device\HarddiskVolume1). Rootkits create rogue symlinks to redirect device access, making this a useful forensic artifact for drive mapping analysis and anti-forensics detection.
