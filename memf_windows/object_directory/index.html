Module object_directory 

Source
Expand description

Windows kernel Object Directory walker.

Enumerates objects in the kernel namespace by walking _OBJECT_DIRECTORY hash tables. The kernel Object Manager stores objects in a tree of directories, each using a 37-bucket hash table of _OBJECT_DIRECTORY_ENTRY chains. This module finds _DRIVER_OBJECT instances under \Driver for IRP dispatch table checking.
