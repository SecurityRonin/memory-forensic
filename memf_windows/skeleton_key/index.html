Skeleton Key backdoor detection (MITRE ATT&CK T1556.001).

The Skeleton Key attack patches LSASS process memory to install a master password that works alongside every user’s real password. Detection involves scanning for known byte patterns in authentication DLLs (msv1_0.dll, kdcsvc.dll, cryptdll.dll, lsasrv.dll) loaded by lsass.exe.

Key indicators:

  • NOP sleds (0x90 repeated) in msv1_0.dll near MsvpPasswordValidate
  • Patched conditional jumps (0xEB replacing 0x75) in kdcsvc.dll
  • Modified RC4 init routines in cryptdll.dll
  • Authentication bypass patches in lsasrv.dll
