PPID spoofing detection (MITRE ATT&CK T1134.004). PPID spoofing detection (MITRE ATT&CK T1134.004).

Checks known Windows system processes against their expected parent process names. When svchost.exe is not a child of services.exe, or lsass.exe is not a child of wininit.exe, it is a strong signal of process injection via PPID spoofing.

The parent-process rule table lives in forensicnomicon::processes so it can be reused across crates without duplication.
