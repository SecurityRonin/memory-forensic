    reader: &ObjectReader<P>,
) -> Result<Vec<TlsCallbackInfo>>
Expand description

Walk TLS directories of all loaded modules in all processes and validate callback addresses.

For each _IMAGE_TLS_DIRECTORY found, extracts the callback array and checks that each entry falls within [module_base, module_base + size). Callbacks that resolve outside this range are flagged as is_outside_module = true.
