Module envvars 

Source
Expand description

Windows process environment variable extraction.

Reads environment blocks from _EPROCESS -> _PEB -> _RTL_USER_PROCESS_PARAMETERS.Environment. The environment block is a UTF-16LE encoded sequence of KEY=VALUE\0 pairs terminated by a double null.
