Module dse_bypass 

Source
Expand description

Driver Signature Enforcement (DSE) bypass detection.

Reads g_CiOptions from ci.dll to detect whether code integrity checking has been disabled. Normal operation has g_CiOptions = 0x6. Attackers clear this to zero (0x0) to allow loading unsigned drivers, which is a key step in many kernel rootkit installs.

Also checks nt!g_CiEnabled when present (older Windows versions).
