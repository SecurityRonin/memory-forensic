Module userassist 

Source
Expand description

UserAssist evidence-of-execution walker.

Windows stores program launch counts and last-run timestamps in the NTUSER.DAT registry hive under Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist\{GUID}\Count. Values are ROT13-encoded file paths with a fixed-size binary data structure containing run count and FILETIME.

The binary value data (72 bytes on Vista+) has the following layout:

  • Offset 4: Run count (u32)
  • Offset 8: Focus count (u32)
  • Offset 12: Focus time in milliseconds (u32)
  • Offset 60: Last run time (u64, FILETIME)
