Windows Desktop and Window Station enumeration.

Window stations and desktops are security boundaries in Windows. Each session has one or more window stations, and each window station contains one or more desktops. Malware sometimes creates hidden desktops to run GUI payloads invisibly — enumerating desktops reveals these hidden execution contexts.

Key forensic indicators:

  • Non-standard window station names (not WinSta0 or Service-0x0-*)
  • Non-standard desktop names on the interactive station
  • Desktops on unexpected interactive stations
