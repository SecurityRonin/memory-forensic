Module com_hijacking 

Source
Expand description

COM object hijacking detection.

Detects when a CLSID has a HKCU\Software\Classes\CLSID\...\InprocServer32 value that overrides the trusted HKCR path, a technique used by malware to load arbitrary DLLs into COM clients without admin privileges.
