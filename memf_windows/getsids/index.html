Module getsids 

Source
Expand description

Process SID enumeration for privilege escalation detection.

Extracts Security Identifier (SID) information for each process, showing which user/group security context a process runs under. Essential for identifying privilege escalation — if a user-spawned process runs as SYSTEM, that is suspicious. Equivalent to Volatility’s getsids plugin. MITRE ATT&CK T1078/T1134.
