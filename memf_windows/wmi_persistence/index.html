WMI persistent event subscription detection.

WMI subscriptions (EventFilter + EventConsumer + FilterToConsumerBinding) are a heavily-used APT persistence mechanism. Subscriptions survive reboots and execute arbitrary commands or scripts when trigger conditions fire.

This module provides:

  • A heuristic classifier for suspicious WMI subscriptions
  • A graceful-degradation walker that returns empty when WMI symbols absent

A full implementation would scan WMI service process heaps (WmiPrvSE.exe / svchost.exe hosting winmgmt) for subscription objects, or parse the WmipBindingTable kernel structure if present.
