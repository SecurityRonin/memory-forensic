Read a null-terminated UTF-16LE wide string (LPWSTR) from a virtual address.

Reads up to MAX_WIDE_CHARS code units (2 bytes each), stopping at the first null (0x0000) code unit. This handles the LPWSTR / PWSTR pointer type common in Windows user-mode structures (as opposed to _UNICODE_STRING which carries an explicit length field).
