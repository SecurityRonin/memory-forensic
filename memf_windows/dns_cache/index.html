Module dns_cache 

Source
Expand description

Windows DNS resolver cache extraction.

The Windows DNS Client service (Dnscache) maintains an in-memory cache of recently resolved DNS records in dnsrslvr.dll. Extracting this cache from memory reveals what domains a system has resolved — critical for identifying C2 infrastructure, data exfiltration endpoints, and lateral movement targets during DFIR triage.

The cache uses a hash table of DNS_HASHTABLE_ENTRY structures, each containing a doubly-linked list of DNS_CACHE_ENTRY records. Each record stores the queried name, record type, TTL, and resolved data.
