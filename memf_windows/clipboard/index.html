Module clipboard 

Source
Expand description

Windows clipboard content recovery from kernel memory.

The Windows clipboard stores data in win32k’s _CLIP structure array on _WINSTATION_OBJECT. Recovering clipboard contents from memory captures passwords, URLs, or commands that were copied before the dump.

Each clipboard entry has a format (e.g., CF_TEXT, CF_UNICODETEXT, CF_HDROP) and an associated data handle. Text formats are decoded and a preview is extracted for forensic analysis.
