Walk all three PEB loader lists for a process and cross-reference them.

Reads the PEB address from the _EPROCESS at eprocess_addr, then walks each of the three LDR module lists, collecting base addresses and DLL names. The results are merged by base address, and each entry is classified as suspicious or benign.

Returns Ok(Vec::new()) if the process has no PEB (e.g. System, Idle).
