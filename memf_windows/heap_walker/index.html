Module heap_walker 

Source
Expand description

Generic process-heap region scanner.

Provides for_each_heap_region, which encapsulates the repeated pattern of walking process VADs, filtering to private readable-writable non-executable heap pages, reading each region, and collecting results. All credential walkers delegate to this function.

for_each_heap_region 🔒
Walk all private, readable-writable, non-executable VAD regions for every process accepted by filter_fn, pass each region’s bytes to scan_fn, and collect the results deduplicated by key_fn.
