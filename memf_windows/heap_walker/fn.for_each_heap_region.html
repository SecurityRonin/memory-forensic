Expand description

Walk all private, readable-writable, non-executable VAD regions for every process accepted by filter_fn, pass each region’s bytes to scan_fn, and collect the results deduplicated by key_fn.

§Arguments

  • reader — kernel-space ObjectReader (kernel CR3 / symbol table).
  • ps_head_vaddr — virtual address of PsActiveProcessHead.
  • filter_fn — returns true for processes whose heap should be scanned.
  • scan_fn — called with (region_bytes, process), returns ≥0 items.
  • key_fn — extracts a deduplication key from each item.

§Returns

A [WalkResult] containing all unique items found, plus a count of VAD regions that could not be read (paged-out or truncated in the dump).
