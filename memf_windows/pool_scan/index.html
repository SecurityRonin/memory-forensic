Module pool_scan 

Source
Expand description

Pool scanner for finding kernel objects by pool tag.

Scans physical memory ranges for pool allocation headers identified by their 4-character ASCII pool tag. This technique finds kernel objects (processes, threads, drivers) independently of linked lists, catching DKOM-hidden objects that have been unlinked from PsActiveProcessHead or similar structures.
