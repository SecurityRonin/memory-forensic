Module debug_registers 

Source
Expand description

Hardware debug register analysis for anti-debug and rootkit detection.

Checks hardware debug registers (DR0-DR3, DR6, DR7) for each thread by reading _KTHREAD.TrapFrame -> _KTRAP_FRAME debug register fields.

Anti-debug malware uses debug registers for hardware breakpoints to detect debuggers, and rootkits use them for stealthy hooking via hardware breakpoint-based execution redirection.

MITRE ATT&CK: T1622 (Debugger Evasion)
