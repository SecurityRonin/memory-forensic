Module win_crashdump 

Source
Expand description

Windows crash dump (.dmp) format provider.

Parses 64-bit Windows crash dumps with _DUMP_HEADER64. Supports run-based (DumpType 0x01) and bitmap (0x02/0x05) layouts.
