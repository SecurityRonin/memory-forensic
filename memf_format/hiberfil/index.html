Module hiberfil 

Source
Expand description

Windows hibernation file (hiberfil.sys) format provider.

Parses hibernation files with PO_MEMORY_IMAGE header signatures: hibr (0x72626968), wake (0x656B6177), RSTR (0x52545352), HORM (0x4D524F48). Eagerly decompresses Xpress LZ77 blocks into a HashMap<pfn, page> for random-access reads. Extracts CR3 from the processor state page.
