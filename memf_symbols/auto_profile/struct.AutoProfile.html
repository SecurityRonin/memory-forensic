pub struct AutoProfile {

Expand description

Automatic symbol profile resolver for Windows kernels.

Locates and loads the correct PDB file given a kernel PDB identity, a PE binary, or a physical memory dump.

Fields§

§cache_dir: PathBuf

Local directory used as the PDB cache.

§network_enabled: bool

Whether a cache miss may fetch the PDB from the network symbol server. Enabled by default; disable (with_network(false)) for air-gapped work.

pub fn with_network(self, enabled: bool) -> Self

Enable or disable network symbol-server download (default: enabled).

Pass false for air-gapped / offline forensic analysis: a cache miss then returns crate::Error::NotFound instead of reaching the network. This is a runtime setting, never a compile-time feature — the capability is always present.

pub fn from_pdb_id(&self, pdb_id: &PdbId) -> Result<Box<dyn SymbolResolver>>

Resolve a SymbolResolver for the given PDB identity.

Checks the local cache first. On a miss, downloads the PDB from the Microsoft symbol server iff network is enabled; otherwise returns crate::Error::NotFound without touching the network.

Extracts the PDB identity from pe_bytes then delegates to from_pdb_id.

pub fn from_dump<P: PhysicalMemoryProvider>( &self, mem: &P, ) -> Result<Box<dyn SymbolResolver>>

Resolve a SymbolResolver by scanning physical memory for the kernel.

Scans mem for ntoskrnl.exe, extracts its PDB identity, then delegates to from_pdb_id.
