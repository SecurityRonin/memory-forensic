Symbol resolution backends for memory forensics.

Provides the SymbolResolver trait and concrete backends:

  • ISF JSON (Volatility 3-compatible symbol tables)
  • BTF (Linux BPF Type Format, kernel 5.2+)

FieldInfo

RebasedResolver
A SymbolResolver decorator that rebases global symbol addresses by a kernel image base virtual address (KVO). Windows ISF stores symbols as RVAs; wrapping the resolver here means every symbol_address caller receives a real virtual address by construction — there is no way to obtain an un-rebased RVA (secure-by-design). Struct/field offsets are image-relative and pass through unchanged.
