BTF (BPF Type Format) symbol resolver.

Parses the BTF section from a vmlinux binary or raw BTF data. BTF provides struct layouts and type information but NOT symbol addresses.

Reference: https://www.kernel.org/doc/html/latest/bpf/btf.html
