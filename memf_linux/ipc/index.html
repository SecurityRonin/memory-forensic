Linux System V IPC object enumeration.

Enumerates shared memory segments, semaphores, and message queues from kernel memory by walking the shm_ids and sem_ids structures. These IPC mechanisms can be used for covert data exchange between processes and are relevant for detecting process collaboration or C2 channels.
