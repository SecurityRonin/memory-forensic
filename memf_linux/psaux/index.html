Module psaux 

Source
Expand description

Detailed process information extraction (Linux ps aux equivalent).

Extracts runtime statistics from each task_struct: CPU state, virtual/resident memory sizes, TTY, process state, nice value. Extends basic process enumeration with data useful for DFIR triage. Identifies zombie processes, stopped processes, and resource anomalies.
