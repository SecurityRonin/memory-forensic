# Privacy Policy

**Effective date:** 2026-01-01

`memory-forensic` is an open-source Rust library and CLI toolkit. It runs entirely on your local machine and does not collect, transmit, or store any personal data, telemetry, or usage information.

## What we collect

Nothing. The software operates offline. No data leaves your machine during normal use.

## Symbol server requests

If you use `memf-symbols` with the built-in symbol server client (`--symbol-server`), your machine sends PDB GUID and age values to Microsoft's public symbol server (`msdl.microsoft.com`). This is equivalent to opening a URL in a browser. Microsoft's [privacy statement](https://privacy.microsoft.com/en-us/privacystatement) applies to those requests.
